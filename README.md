# 📈 StockBoard: 종합 주식 현황판 (Stock Dashboard)

> **"주식 투자의 첫걸음, 시장의 흐름을 한눈에 파악하는 것부터 시작하세요!"**  
> 본 프로젝트는 국내/해외 주식 및 ETF 시장의 데이터를 시각화하여 투자자가 현명한 결정을 내릴 수 있도록 돕는 웹 기반 대시보드입니다.

---

## 🌟 프로젝트 소개
주식을 처음 시작하면 수많은 숫자와 복잡한 용어에 당황하기 쉽습니다. **StockBoard**는 복잡한 시장 데이터를 직관적인 UI(다크 모드)로 제공하여, 초보 투자자도 현재 시장이 어떻게 흘러가는지, 내가 관심 있는 기업의 실적은 어떤지 쉽게 확인할 수 있도록 설계되었습니다.

## 🚀 주요 기능 (초보자를 위한 가이드)

### 1. 실시간 시장 지수 티커
페이지 상단에 흐르는 바를 통해 전 세계 시장의 온도를 즉시 확인할 수 있습니다.
*   **KOSPI (코스피):** 대한민국 대표 기업들이 모인 시장의 성적표입니다.
*   **NASDAQ (나스닥):** 애플, 마이크로소프트 등 미국의 혁신 기술주들이 모인 시장입니다.
*   **S&P 500:** 미국에서 가장 우량한 500개 기업을 모아놓은 지수로, 미국 경제의 척도입니다.
*   **환율 (USD/KRW):** 1달러를 사는 데 필요한 우리 돈입니다. 해외 주식 투자 시 매우 중요합니다.

### 2. 시장 카테고리별 탐색
*   **국내 주식:** 삼성전자, 현대차 등 우리나라 시가총액 상위 기업 리스트를 제공합니다.
*   **해외 주식:** 전 세계 1등 기업인 애플, 엔비디아 등 미국 우량주를 확인합니다.
*   **ETF (KODEX/TIGER):** 개별 주식을 고르기 힘들 때 여러 주식을 묶어서 투자하는 상품입니다. 삼성자산운용(KODEX)과 미래에셋자산운용(TIGER)의 인기 상품을 모아두었습니다.

### 3. 인터랙티브 주가 차트
*   종목을 클릭하면 실시간 가격 흐름이 차트로 나타납니다.
*   **상승은 빨간색, 하락은 파란색**으로 표시되어 직관적인 확인이 가능합니다. (한국 거래소 기준)

### 4. 핵심 재무제표 요약 (기업 건강검진)
기업이 돈을 잘 벌고 있는지 3년치 데이터를 보여줍니다.
*   **매출액/영업이익:** 기업이 장사를 얼마나 잘했는지 보여줍니다.
*   **ROE (자기자본이익률):** 내 돈을 투자해서 기업이 얼마나 효율적으로 수익을 냈는지 나타내는 지표입니다. (높을수록 좋음)
*   **PER (주가수익비율):** 기업의 수익 대비 주가가 싼지 비싼지 알려줍니다. (낮을수록 저평가)

## 🛠 기술 스택 (Tech Stack)
*   **Frontend:** HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript
*   **Library:** [Chart.js](https://www.chartjs.org/) (데이터 시각화)
*   **Icon:** [FontAwesome](https://fontawesome.com/)
*   **Data Source:** Google Finance 실시간 크롤링 데이터 반영 (2026-04-17 기준)

## 💻 실행 방법
별도의 설치 과정 없이, 웹 브라우저만 있으면 실행 가능합니다.

1.  본 저장소를 `Clone` 하거나 `.zip` 파일로 다운로드합니다.
2.  `stock_index.html` 파일을 크롬(Chrome)이나 엣지(Edge) 브라우저로 엽니다.
3.  화면 왼쪽 메뉴를 클릭하며 시장 데이터를 탐색하세요!
